08 - Data Models: Build From Scratch

Main Purpose

  • Create your own data model from scratch
  • Learn about the components that form a data model

Guide to Completing 08

This tutorial shows you how to create a new data model by building your own python files that utilize marshmallow, elasticsearch mappings, JSONSchema mappings, loaders, serializers, and more. This module has a lot of working with uncommenting Python code. If you are new to Python, and are not familiar with their indentation and commenting rules, it is integral you be familiar with these two concepts in particular so you know what to uncomment. You may view this for help.

****Warning For This Tutorial
Use the Step 1: Bootstrap exercise with extreme caution. It will get rid of all previous changes made to files/folders in any tutorials prior.
