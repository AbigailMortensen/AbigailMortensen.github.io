About This Documentation

How to Use This Documentation

  • This guide is best suited to those unfamiliar with the terminologies/technologies mentioned in the Invenio tutorials.
  • This documentation was written so that the reader may look at tutorial sections individually without having to read all the info for the tutorials before. For this reason, there is some repetitive information throughout.
  • Subscripted numbers denote an id for a hyperlink, where you may find the full link in the Referenced Link section of this documentation if the hyperlink doesn't work.
  • This guide would be well suited to those wanting to read about someone else's experience with the tutorials before performing the tutorials themselves.
  • The last section, Useful Information, is a sort of summation of the information provided in later tutorials.

Background on This Documentation

  • This guide contains my notes and thoughts as I went through the Invenio tutorials.
  • Prior to the tutorials, I was a novice in my understanding of Invenio and the software it was built with.
  • There are two tutorials I was not able to successfully complete; my notes are not perfect but I have tried to make them as helpful as possible as I have included advice I wished I had known prior to each tutorial.

Supplementary Materials

I highly recommend reading supplementary materials either prior to, or in conjunction with, the completion of each tutorial. It is not until later tutorials that you will receive a more detailed explanation of why you are doing each step. I will be adding links to reading materials for each tutorial if applicable.
