04 - Running Invenio

Main Purpose

  • Learn to navigate in the development server through the terminal
  • View entry points and Celery worker
  • Learn how to use the "invenio command"

Guide to Completing 04

The "invenio" command can only be used in a python virtual environment, which you may run by typing "pipenv shell" while in the directory ~/src/my-site. You will need to be familiar with how to access this virtual environment in later tutorials. When in this environment, you may pull up a Python shell or you may access the invenio shell with the command "invenio shell". If you are unfamiliar with some of this terminology, check out these links:

Python shell/REPL
Python virtual environments

****Why do we access the Python shell and Invenio shell?
Within a Python shell, you may write Python code that is instantly executed but will be erased once you exit the shell (via a technology called REPL, Read-Evaluate-Print-Loop) . However, this Python shell comes as a blank slate for you to practice/test Python code in, with nothing pre-installed. The invenio shell is a Python shell complete with pre-loaded configurations and extensions that allow you perform high-level APIs without needing to know low-level details. You will learn more about the pragmatic uses of the invenio shell in later tutorials.
