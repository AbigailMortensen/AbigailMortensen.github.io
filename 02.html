02 - Tour of Invenio

Main Purpose

  • Become acquainted with the invenio webpage and what it has to offer for regular users and administrators
  • Create a regular account online and create an administrative account through REST API
  • Set records with a REST API POST request, retrieve records with a REST API GET request

Guide to Completing 02

****If you are new to localhosts/development servers
If you choose to close the development server(by closing the tab running the development server) on your localhost, no information that you input into the server previously will be saved. For instance, if you created an administrator user account, this account would be deleted once the server it was created on is shut down.

****If you are new to the command line
To complete step 4, open up a separate terminal window from the window that is running the web server from the last tutorial. In this new window, go to the my-site directory by typing "cd ~/src/my-site". The command you type is from "curl -k --header" to "prettyprint=1". Note, the '\' character allows you to press enter without executing the command--if you don't want to press enter to visually format the command, you may omit the '\'.

***** What is the OAI-PMH endpoint and why do we care about it?
OAI-PMH is a protocol for metadata harvesting. OAI-PMH defines a "verb" (a command) called "Identify", which we use in Step 8 to access descriptive information about the repository. See section 4.2 of this link for more information on "Identify" or OAI-PMH.
