12 - Managing Access

Main Purpose

  • Creating access permissions

Guide to Completing

We are creating a permissions factory module that defines a function that searches for the user's id. This permissions factory is then imported into records configuration file so that to edit, view, or delete a JSON record of a file you must verify that you are its owner. In the records config file you edit both the REST API endpoints (so you cannot view it in the JSON format) and the UI endpoints (so you cannot view information in the user interface either).

As a heads up, the completion of this tutorial will make it more difficult to view records or add records through the REST API or website UI, so make sure to use just the usernames and passwords provided by the tutorial so you don't forget login information and lose access to record information.
